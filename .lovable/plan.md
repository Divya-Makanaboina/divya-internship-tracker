

# Internship Application Tracker Dashboard

## Overview
A comprehensive dashboard to help you track and analyze your internship applications, stored locally in your browser.

---

## Features

### 1. Dashboard Header & Summary Cards
- **Total Applications** - Count of all applications
- **Active Applications** - Applications in progress (Applied + Interview)
- **Success Rate** - Percentage of offers received
- **Pending Responses** - Applications awaiting response

### 2. Visual Analytics Charts
- **Status Breakdown** - Pie/donut chart showing distribution across Applied, Interview, Offer, Rejected
- **Applications Over Time** - Bar chart showing applications by week/month
- **Department Distribution** - Bar chart showing which departments you're targeting most

### 3. Application Data Table
Columns:
- Company Name
- Job Position
- Department
- Application Status (with color-coded badges)
- Date Applied
- Notes (expandable)
- Actions (Edit, Delete)

Features:
- Sort by any column
- Filter by status
- Search by company name or position
- Inline editing

### 4. Add/Edit Application Form
- Modal form to add new applications or edit existing ones
- Date picker for application date
- Dropdown for status and department
- Text area for notes

### 5. Data Persistence
- All data saved to browser's localStorage
- Data persists across sessions on the same device/browser

---

## Design Style
- Clean, modern dashboard layout
- Color-coded status badges (blue for Applied, yellow for Interview, green for Offer, red for Rejected)
- Responsive design that works on desktop and mobile
